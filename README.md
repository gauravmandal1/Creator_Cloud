## Run Locally

Clone the project

```bash
  git clone https://github.com/gauravmandal1/Creator_Cloud
```

Go to Creator_cloud directory

```bash
  cd Creator_cloud
```

Install dependencies

```bash
  npm install i
```

Start the server

```bash
  npm start
```
